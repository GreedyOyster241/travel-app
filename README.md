# ✈️ TravelPath

A cross-platform desktop travel planning app built with **Electron** and **React** that helps you find the best way to get anywhere — whether by car, plane, or boat — and discover the best deals along the way.

---

## 🧭 Overview

TravelPath compares all available travel modes between your origin and destination, showing you estimated times, costs, and helping you plan everything from flights to hotel stays to local attractions — all in one place.

---

## ✨ Features

- **Multi-modal travel comparison** — Compare driving, flying, ferries, and more side by side (time, cost, carbon footprint)
- **Fastest route planning** — Get turn-by-turn driving directions and estimated travel times
- **Flight search** — Browse available flights with real-time pricing
- **Hotel recommendations** — Find the best hotel deals near your destination based on your travel method
- **Points of interest** — Discover top attractions, restaurants, and things to do once you arrive
- **Secure API key handling** — Keys are stored safely in Electron's main process, never exposed to the renderer

---

## 🔌 APIs Used

| API | Purpose |
|-----|---------|
| [Rome2rio API](https://www.rome2rio.com/documentation/) | Multi-modal travel options (driving, flying, ferry, train) |
| [Google Maps / Directions API](https://developers.google.com/maps/documentation/directions) | Driving routes, estimated time, and distance |
| [Amadeus API](https://developers.amadeus.com/) | Flight search and pricing (free tier available) |
| [Booking.com Affiliate API / RapidAPI Hotels](https://rapidapi.com/apidojo/api/hotels4/) | Hotel deals near your destination |
| [Google Places API](https://developers.google.com/maps/documentation/places/web-service) | Points of interest and things to do |

## 🖥️ Tech Stack

- **Electron** — Cross-platform desktop shell
- **React** — UI framework
- **Node.js** — Main process / API key management

---

## 🗺️ App Flow

1. User enters an **origin** and **destination**
2. Rome2rio returns all available travel modes with estimated cost and time
3. User selects a travel mode (fly, drive, boat, etc.)
4. App displays:
   - Detailed route or flight options
   - Hotel deals near the destination
   - Top attractions and things to do in the area

---

## 📱 Screens

- **Search** — Enter origin, destination, and travel dates
- **Results** — Side-by-side comparison of all travel modes
- **Detail** — Map view, hotel listings, and local attractions for the chosen route
